 giải nén 3 file mặc định của game ( nhớ copy lưu lại bản sao )
lua.package
skin.package
text.package

copy đè các file auto vào file mặc định
riêng file text interface.idres khá nặng nên mình không đính kèm
các bạn dùng notepad++ thêm các dòng trong file text/edit.txt vào là được

Nếu vào game lỗi giao diện thì mở thư mục skin đọc hướng dẫn

dùng phần mềm unpack nén lại và dùng auto

thư mục autofree copy vào thư mục res ( cùng nơi với file lua.package ) để có auto thư sinh

cách giải mã file .lua mặc định của game để so sánh cũng như chỉnh sửa cho 1 số bạn thích
cài đặt java
tải file unluac 
link https://sourceforge.net/projects/unluac/
về đổi tên thành unluac.jar

dùng notepadd tạo file unluaall.bat với nội dung

for /f "TOKENS=*"  %%s IN ('dir /b /o /s *.lua') do java -jar unluac.jar "%%s" > "%%s.txt"

bây giờ muốn giải mã file lua nào chỉ cần copy 2 file 
unluac.jar + unluaall.bat 
vào thư mục chứa file lua chạy file .bat để giải mã toàn bộ các file .lua có trong thư mục

các file chỉnh sửa ở vị trí nào, khác gì với file mặc định mình đã ghi chú khá rõ, mọi người có thể tìm kiếm dòng ghi chú ( -- ) 2 dấu trừ
